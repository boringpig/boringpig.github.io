Blog搭配Hexo主題

2016-07-27
Framework

安裝

  • NodeJS安裝
  • Git安裝
  • 如果不行npm install hexo-cli -g 可試看看sudo npm install -g -unsafe-perm hexo 就可以

    初始設定

  • 先建立一個folder
  • hexo init <folder> 初始化就有hexo的預設主題
  • hexo s 在本機端即可預覽
  • hexo new 建立一個新的文章

    環境設定

  • 先建立 xxx.github.io 的 repository
  • npm install hexo-deployer-git --save用來發佈到github的套件

  • 根目錄.config.yml設定

  • 設定完成後輸入 hexo d 發佈到github

  • 下次再發佈須先輸入 hexo clean 才可以發佈

    更換主題

  • cd <folder>/themes 然後 git clone 別人的repo

  • 再到外部.config.yml

  • hexo clean、hexo d 即可更換
