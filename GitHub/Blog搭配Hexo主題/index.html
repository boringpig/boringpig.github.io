Step 1

先安裝Hexo所需要的套件，Hexo 官方中文文件寫的非常詳細。

Step 2

先 cd 到該目錄的前一個目錄底下


Step 3

重新整理之後還是沒變，因為沒有push上去

Step 4

先用 hexo serve 在本機端先預覽，看到這畫面代表已經套用 Hexo 的預設主題了。

Step 5

安裝這個套件是可以將 hexo 的 markdown 文件的編譯成 html 到 GitHub 上

Step 6

Step 7
