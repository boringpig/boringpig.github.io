Google Maps基本認識

2016-09-12
Google API

現在很多商業或是部落格的網站，強調地方的所在位置，不僅給地址還會用地圖給別人看，讓人可以清楚知道相對位置或是附近的店家。我覺得這個API蠻容易學習的，w3school 網站真的講得很棒。

Type 介紹

Google map 顯示的類型

  • ROADMAP : 2D map (default)
  • SATELLITE : photographic map 衛星上拍照的圖
  • HYBRID : photographic map + roads and city names
  • TERRAIN : 包括山、河流地圖

Overlay 介紹

Overlay 在 google map 是指在眾多地方的經緯度集合而成的形狀

  • Marker : 單一個地方就會顯示預設箭頭的形狀。
  • Polyline : 兩個以上的地方就會連成一條直線。
  • Polygon : 多個直線組成一個”封閉”迴圈的形狀。
  • Info Windows : 單一個地方顯示訊息框的內容，可搭配 marker 使用。
  • Custom overlays : 除了 google map 預設的也可以客製化形狀。

Control Panel介紹

Default Panel :

  • Zoom : 地圖上右下角有個 “+/-“ buttons 來控制地圖的遠近
  • Pan : 地圖上有個手的圖案用來控制地圖平移
  • MapType : 地圖上左上角可以讓你選擇地圖的類型，像是衛星圖、地形圖等等
  • Street View : 地圖上右下角 Pegman icon 用來拉到地圖上顯示街道圖

More Control :

  • Scale : 顯示地圖上比例尺
  • Rotate : 使用街道圖時，顯示 circular icon 讓你可以旋轉地圖的位置
  • Overview Map : 拉近地方位置，可以顯示更寬廣的 viewpoint 像是河流、山脈等著名景點
